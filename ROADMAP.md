# Roadmap

## Current Version (v1.3.0)

### Completed Features
- Audio downloads (M4A) with Apple AAC encoding
- Video downloads (MP4) with Apple AAC audio
- iPod Format option for iTunes-compatible video (H.264, 640x480, .m4v)
- Automatic artwork embedding from YouTube thumbnails
- Metadata extraction (artist, title, album)
- Playlist support (download all tracks from a YouTube playlist)
- iPod detection and direct copy
- Queue management with individual stop buttons
- Clear completed downloads
- Add to queue while downloads are running
- Organized folder structure (Audio/ and Videos/ subfolders)
- Configurable download location

---

## Planned Features

### Bulk URL Import
- Add a toggle to switch between single URL and bulk mode
- Text area for pasting multiple URLs (one per line)
- Parse and add all URLs to the download queue at once
- Support for mixed video/audio URLs in bulk mode

### Custom Collections

#### Mixtapes (Various Artists)
- Create custom playlists with tracks from different artists
- Name the mixtape
- Reorder tracks via drag-and-drop
- Export as a folder with numbered tracks
- Option to embed mixtape name as album metadata
- Save/load mixtape configurations

#### Albums (Single Artist)
- Create or recreate albums from individual tracks
- Set album name, artist, and year
- Add album artwork (from URL or local file)
- Automatic track numbering
- Export as organized album folder
- Embed consistent album metadata across all tracks
- Template support for recreating known albums
